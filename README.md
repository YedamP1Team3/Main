# 프로젝트명 : 발달장애지원 프로그램

## 개요
 발달장애인에 특화되지 않은 복지 서비스 구조와, 
 직접 이용이 어려운 이용자를 고려하지 못한 참여 방식에서 좀 더 친화적으로 
 접근 할 수 있는 방식이 필요하다 생각했음 또한
 보호자가 함께 참여할 수 있는 구조를 통해 
 서비스 접근성과 이용 편의성을 개선할 수 있음

## 팀원
 - 신재성(팀장) - 배포
 - 송민규(부팀장) - git관리
 - 김덕모 - 품질 - 환경관리
 - 김태윤 - 영업 - DB관리
